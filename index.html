CONCEPT

リーガル・パレットとは？

「法律事務所」と一言で言っても、企業法務、刑事、街弁、国際など、 先生方の働き方や事務所のカラー（個性）は千差万別。
私たちは、まだ自分の色を知らない学生が、色とりどりの事務所を実際に訪ね、 自分だけのパレットを描いていく場所です。
見て、知って、選べる法曹キャリアへ。

MISSION

机の上から、実務の前へ。自分のキャリアを、自分のものに。

ただ勉強するだけの場所から実務のリアルに飛び込み、誰かに決められたルートではなく「自分自身の意志でキャリアを掴み取る」ことを掲げる。

VISION

誰もが、理想の法曹像をその胸に、確信を持って歩みを進められる世界

「訪問」という体験を通して自分自身を深く見つめ直し、誰もが自分だけの確固たる理想像に出会える未来。そのゴールが胸にあるから、一歩一歩に確信を持って進める。

VALUES

成長を、楽しむ

Growth Mindset

司法試験や就活を「課題」ではなく、実務に触れる全機会を成長のチャンスとして楽しむ。

壁を、超える

Cross Border

大学の垣根、既修・未修の壁、「学生だから」という限界をフラットに越え、関西全体で高め合う。

自分ゴト化する

Ownership

サークルもキャリア選択も他人任せにしない。当事者意識を持ち、自分の人生をドライブする。

FOR LAW FIRMS

未来の法曹の「色」を見つける場所。

就活の早期化が進み、法曹界の採用競争も激化する一方で、事務所の「真の魅力」が学生に伝わりきらず、 入所後に「イメージと違った」というミスマッチが生まれることも。
リーガルパレット関西は、「対話」を通じてこの課題を解決します。

早期接点によるリード

事務所訪問というカジュアルな場で、早い時期から学生に貴所の存在と魅力を知ってもらえます。

解像度を高める（ミスマッチ防止）

直接お話を聞くことで学生が貴所への深い理解を持ち、入所後の納得感（定着率）を高めることができます。

認知度向上・未来のパートナーとの出会い

貴所の「個性」を感度の高い学生へ直接アピールし、将来を担う人材と早期に関係を築けます。

次世代を担う法曹志望の学生たちに、貴事務所の「色」を直接伝えてみませんか。まずは、貴所の「知りたいこと」「学生に伝えたいこと」をぜひお聞かせください。

TYPE DIAGNOSIS

法曹タイプ診断

あなたはどのタイプ？ 4つのカラーから、自分らしい法曹のかたちのヒントを見つけてみましょう。

TYPE 01

論理派ストラテジスト

Logical Strategist

事実と論理を積み上げ、最適な一手を導き出すのが得意。渉外・企業法務など、緻密な戦略が求められる分野で力を発揮します。

TYPE 02

人間派サポーター

Empathetic Supporter

相手の気持ちに寄り添うことが得意。家事事件や一般民事など、依頼者に寄り添う分野でその力を発揮します。

TYPE 03

挑戦派パイオニア

Challenging Pioneer

前例のない分野にも臆せず飛び込む行動力の持ち主。ベンチャー支援や新領域の開拓に向いています。

TYPE 04

専門派スペシャリスト

Focused Specialist

一つのことをとことん突き詰める集中力が強み。知的財産や税務など、専門特化分野で活躍します。

FAQ

よくある質問

活動頻度はどのくらいですか？

月に2〜3回、法律事務所への訪問や座談会、サークル内の勉強会などを実施しています。

ロースクール生でないと参加できませんか？

基本的には、ロースクール生を対象としていますが、法曹に関心のある学部３・４年生様であれば、代表と面談後入会を許可する場合があります。

参加できるロースクールは決まっていますか？

関西エリアのインカレサークルのため、複数のロースクールから学生が参加しています。所属ロースクールは問いません。

事務所訪問の際の服装に決まりはありますか？

基本的にはリクルートスーツを推奨していますが、訪問先によって案内が異なる場合は事前にお伝えします。

就職活動のためのイベントなのでしょうか？

就職活動を目的とした団体ではありません。あくまで多様な法曹キャリアと出会い、視野を広げることを大切にしています。

参加費用はかかりますか？

基本的に参加費は無料です。一部サークル内で懇親会をする場合には、飲食代等を負担いただく場合がございます。詳細はお問い合わせフォームよりご確認ください。

授業や試験（定期試験・司法試験）との両立はできますか？

はい、十分に両立可能です。活動は原則として月2〜3回、平日夕方や休日など、学業の負担にならないスケジュールで調整しています。試験期間前などは活動を完全にオフにするなど、学業最優先で参加できる環境です。

文系・理系や、法学部出身でない「未修者コース」でも参加できますか？

もちろん大歓迎です！関西エリアの複数のロースクールから、法学部出身の既修者だけでなく、他学部出身の未修者の方も多数参加しています。バックグラウンドに関わらず、多様な視点を持ったメンバーが集まっています。

法律事務所の訪問では、具体的にどのようなことをするのですか？

事務所のパートナー弁護士や若手弁護士の方々へのインタビュー、執務室やフロアの見学、日々の業務内容やキャリアについての座談会などを中心に行っています。単なる見学ではなく、実際のリアルなお話を直接伺うことができる貴重な機会です。

個人情報の取り扱いや、訪問時の守秘義務はどうなっていますか？

法律事務所を訪問する性質上、業務で知り得た情報や、事務所内での機密事項について外部に漏らさないことなど、社会人としての最低限の守秘義務を徹底していただいています。初回参加時にガイドラインをご案内します。
